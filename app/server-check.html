⚠️ Server Required

This web application requires a local server to function properly.

Launch Server:

Windows Users:
1. Locate "start_server.bat" in your project folder
2. Double-click to run it
3. Keep the terminal window open
Mac/Linux Users:
1. Locate "start_server.sh" in your project folder
2. Open terminal in project folder
3. Run these commands:
chmod +x start_server.sh
./start_server.sh

After Server Starts:

Access the application at:

http://localhost:8000/app/replit.html
Open Application
